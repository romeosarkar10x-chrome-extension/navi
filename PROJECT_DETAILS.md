Navi is a browser extension that adds a persistent, resizable sidebar to any webpage. It serves as an AI-powered copilot that can chat with the user, see and understand the current page, and take autonomous actions on the browser — forming a full agent loop. Think of it as a smart assistant that lives in your browser and can actually _do_ things, not just talk.

The extension supports multiple LLM providers: cloud-based (Claude via Anthropic API) and local models (via LM Studio's OpenAI-compatible endpoint at localhost). Users can switch between providers seamlessly.
